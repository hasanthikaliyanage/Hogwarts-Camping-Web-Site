# 🏕️ Hogwarts Camping Web Site

A front-end web project built using **HTML, CSS, and JavaScript**, designed to showcase a camping-themed website inspired by Hogwarts.  
This project focuses only on **UI/UX design** and does not include backend or database functionality.

---

## ✨ Features

- Responsive and clean UI design  
- Multiple pages (About, Contact, Activities, Forms, etc.)
- Styled using CSS and Bootstrap  
- Interactive elements using JavaScript  
- Image galleries and themed visuals

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **JavaScript** – Basic interactivity  
- **Bootstrap 5** – Responsive design and components
